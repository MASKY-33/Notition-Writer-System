# Notition Writer System
This system allows the user to write notes, read all saved notes, or reset the file.
It demonstrates simple file handling with three basic operations.

## Features
- Option 1: Write a new note to notition.txt
- Option 2: Read and display all saved notes
- Option 3: Reset the file by overwriting it
- Uses append mode for writing and write mode for resetting
- Straightforward, single‑step actions

# How to use
Choose an option:
1 → Write a note
2 → Read all notes
3 → Reset the file

Each action directly interacts with notition.txt.
No loops, no complex logic — just simple file operations.

## Learning Purpose
Practice with:
- writing to files
- reading files
- resetting/overwriting files
- basic menu‑based interaction
